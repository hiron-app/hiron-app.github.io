読書の旅

Dokusho no Tabi

最終更新日：2026年6月29日　／　本プライバシーポリシーは日本語を正式言語とします。

はじめに

「読書の旅」（以下、「本アプリ」）をご利用いただきありがとうございます。本アプリは、読んだ本をムードとデータで美しく振り返ることを目的として開発された個人アプリです。

本プライバシーポリシーは、本アプリがどのような情報を扱い、それをどのように利用するかを説明するものです。本アプリをご利用になる前に、本ポリシーをよくお読みください。

1. 収集する情報

1-1. アプリ内でお客様が入力・記録する情報

本アプリは、以下の情報をお客様のデバイス上に保存します。お客様が iCloud 同期を有効にしている場合は、Apple の iCloud（お客様の Apple アカウント）にのみ保存・同期されます。

  • 本の情報：タイトル・著者・出版社・ISBN・表紙画像・ジャンル
  • 読書ステータス：「読みたい」「読書中」「読了」の状態、読み始め日・読了日
  • 記録：星評価・読後感（感情）・メモ・読了までの歩み
  • 旅のカード：読了時に記録した気持ち・カードのテーマ設定
  • 図書館設定：お客様が登録したお近くの図書館
  • 読書ポモドーロ：集中した読書時間の記録
  • アプリ設定：外観モード・カラーテーマ・通知設定・オンボーディング完了フラグ
これらのデータはすべて、お使いのデバイス（および、有効にしている場合は iCloud）に保存されます。開発者がこれらの内容にアクセスすることはできません。

1-2. 外部サービスとの通信で発生する情報

本アプリは、機能提供のために以下の外部サービスと通信します。いずれもお客様を個人として識別できる情報は送信されません。

  • openBD ／ 楽天ブックス書誌検索 API：書誌情報・表紙画像の取得のため、ISBN または検索キーワードを送信します
  • カーリル（calil.jp）：図書館の蔵書・貸出状況の確認のため、ISBN および対象図書館の識別情報を送信します
  • iCloud（Apple Inc.）：端末間でのデータ同期のため、お客様の Apple アカウントを通じて通信します
  • App Store / StoreKit（Apple Inc.）：「読書の旅 Pro」の購入・復元のため、Apple が管理する購入情報を取得します
  • PostHog（プロダクト分析）：アプリの改善を目的に、匿名の利用イベントを PostHog のサーバー（米国）へ送信します（詳細は「1-3」をご覧ください）

各サービスのプライバシーポリシーについては、各社のサイトをご確認ください。

  • Apple のプライバシーポリシー：https://www.apple.com/legal/privacy/
  • カーリル：https://calil.jp/
  • PostHog のプライバシーポリシー：https://posthog.com/privacy

1-3. 利用状況の解析（PostHog）

本アプリは、機能改善のために PostHog（プロダクト分析・米国）を用いて匿名の利用イベントを計測します。送信されるのは、以下のような個人を特定できない情報のみです。

  • どの機能が使われたかを示すイベント名（例：アプリ起動、本の登録・読了、統計の閲覧、旅のカードの作成・シェア、課金導線の通過など）と、それに付随する数値
  • アプリのバージョン・OS バージョン・デバイスの機種・国／地域などの一般的な技術情報
  • アプリがランダムに割り当てた匿名 ID

これらの情報について、本アプリは次を遵守します。

  • 氏名・メールアドレス・端末の固定 ID（IDFA 等）など、お客様を個人として特定できる情報は送信しません
  • 送信される ID はアプリが割り当てた匿名 IDであり、広告識別子ではありません
  • 広告目的の利用や、他社アプリ・サイトをまたいだ行動追跡（トラッキング）は一切行いません
  • 登録した本の情報・メモ・読後感などの個人データそのものは送信されません

2. 情報の利用目的

情報の種類利用目的
本の情報・読書ステータス・記録本棚・読書記録・統計の表示
星評価・読後感・読了の歩み統計（感情マップ等）と旅のカードの生成
図書館設定蔵書・貸出状況の確認
StoreKit 購入情報「読書の旅 Pro」機能の有効化
匿名の利用イベント（PostHog）アプリの不具合把握・機能改善・利用状況の分析

3. 情報の共有・第三者提供

本アプリは、以下の場合を除き、お客様の情報を第三者と共有しません。

  • 法的要求：法律または法的手続きにより開示が必要な場合
お客様のデバイス（および iCloud）に保存されたアプリデータは、開発者を含む第三者がアクセスすることは一切できません。

4. データの保存と削除

  • アプリ内のデータは、お客様のデバイス上（および有効にしている場合は iCloud）に保存されます
  • アプリをアンインストールすると、デバイス上のデータが削除されます
  • 各本の詳細画面から、登録した本・記録を個別に削除できます
  • iCloud に保存されたデータは、Apple アカウントの設定からも管理・削除できます
  • 開発者はお客様のデータにアクセスする手段を持っていません

5. 広告・トラッキングについて

本アプリは、広告を一切表示しません。広告識別子（IDFA）の利用や、他社アプリ・サイトをまたいだ行動追跡（トラッキング）は一切行いません。

アプリ改善のため、プロダクト分析サービス PostHog を用いて匿名の利用イベントを収集します（詳細は「1-3」参照）。これは個人を特定しない統計的な分析のみを目的とし、収集される ID はアプリが割り当てた匿名 ID であり、広告識別子（IDFA）ではありません。

6. OS 権限について

本アプリは、機能提供のため以下の OS 権限を利用します。許可は任意であり、許可しなくても基本機能は利用できます。

  • カメラ（Camera）：バーコード（ISBN）スキャンによる本の登録
  • 写真（Photo Library）：「旅のカード」を画像として書き出し・保存する場合
  • 通知（Notifications）：読書のリマインドやポモドーロの通知

これらの権限は、お客様が該当機能を使用した時のみ発生します。取得した内容は端末内のアプリデータとして扱われ、本ポリシーに記載のない外部送信は行いません。

7. セキュリティ

  • 入力データはお客様のデバイス上（および iCloud）に保存されます
  • 外部通信は本ポリシー「1-2」に記載のもののみで、いずれも HTTPS 通信を使用します
  • お客様を個人として識別できる情報を外部に送信することはありません

8. 本ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。重要な変更が生じた場合は、アプリのアップデートを通じてお知らせします。最新のポリシーは本ページ（サポートサイト）でご確認いただけます。

9. お問い合わせ

本プライバシーポリシーに関するご質問・ご意見は、開発者ホームページのお問い合わせ先からご連絡ください。

サブスク

Subsuku

最終更新日：2026年6月3日　／　本プライバシーポリシーは日本語を正式言語とします。

はじめに

「サブスク」（以下、「本アプリ」）をご利用いただきありがとうございます。本アプリは、毎月のサブスクリプション支出をひとめで把握し、納得できる支出に整えることを目的として開発された個人アプリです。

本プライバシーポリシーは、本アプリがどのような情報を扱い、それをどのように利用するかを説明するものです。本アプリをご利用になる前に、本ポリシーをよくお読みください。

1. 収集する情報

1-1. アプリ内でお客様が入力する情報

本アプリは、以下の情報をお客様のデバイス上にのみ保存します。これらの情報は外部サーバーには送信されません。

  • サブスクリプション情報：サービス名・プラン名・金額・通貨・請求周期・カテゴリ・契約開始日・次回請求日
  • カスタムロゴ画像：お客様がフォトライブラリから選択した画像（サブスク PRO 機能）
  • 利用状況：最終利用日（お客様が記録した場合）
  • 価値配点：「心の充足」「自分磨き」「生活の土台」「効率化」「つながり」の 5 領域に振り分けた点数
  • 解約履歴：解約したサブスクの名前・金額・解約日（節約実績表示のため）
  • アプリ設定：通知タイミング・通知時刻・通貨デフォルト・外観モード・カラーテーマ・オンボーディング完了フラグ
これらのデータはすべて、お使いのデバイスの SwiftData（ローカルストレージ）および UserDefaults に保存されます。

1-2. 外部サービスとの通信で発生する情報

本アプリは、機能提供のために以下の外部サービスと通信します。いずれもお客様を個人として識別できる情報は送信されません。

  • iTunes Search API（Apple Inc.）：サービスロゴ画像の取得のため、サービス名のみを送信します
  • Exchange Rate API（open.er-api.com）：為替レートの取得のため、GET リクエストのみを送信します（送信内容なし）
  • App Store / StoreKit（Apple Inc.）：サブスク PRO の購入・復元のため、Apple が管理する購入情報を取得します
  • PostHog（プロダクト分析）：アプリの改善を目的に、匿名の利用イベント（例：アプリ起動、サブスクの登録・編集・削除、画面・機能の利用、通知許可の結果、課金導線の通過など）を PostHog のサーバー（米国）へ送信します（詳細は「1-3」をご覧ください）

各サービスのプライバシーポリシーについては、各社のサイトをご確認ください。

  • Apple のプライバシーポリシー：https://www.apple.com/legal/privacy/
  • PostHog のプライバシーポリシー：https://posthog.com/privacy

1-3. 利用状況の解析（PostHog）

本アプリは、機能改善のために PostHog（プロダクト分析・米国）を用いて匿名の利用イベントを計測します。送信されるのは、以下のような個人を特定できない情報のみです。

  • どの機能が使われたかを示すイベント名（例：アプリ起動、サブスクの登録・編集・削除、一覧の閲覧、通知許可の結果、課金導線の通過など）と、それに付随する数値
  • アプリのバージョン・OS バージョン・デバイスの機種・国／地域などの一般的な技術情報
  • アプリがランダムに割り当てた匿名 ID

これらの情報について、本アプリは次を遵守します。

  • 氏名・メールアドレス・端末の固定 ID（IDFA 等）など、お客様を個人として特定できる情報は送信しません
  • 送信される ID はアプリが割り当てた匿名 IDであり、広告識別子ではありません
  • 広告目的の利用や、他社アプリ・サイトをまたいだ行動追跡（トラッキング）は一切行いません
  • 入力したサブスク情報・金額・カスタムロゴなどの個人データそのものは送信されません

3. 情報の共有・第三者提供

本アプリは、以下の場合を除き、お客様の情報を第三者と共有しません。

  • 法的要求：法律または法的手続きにより開示が必要な場合
お客様のデバイスに保存されたアプリデータ（サブスク情報・カスタムロゴ・履歴等）は、開発者を含む第三者がアクセスすることは一切できません。

4. データの保存と削除

  • アプリ内のすべてのデータは、お客様のデバイス上にのみ保存されます
  • アプリをアンインストールすると、デバイス上のすべてのデータが削除されます
  • 設定画面の「全データをリセット」から、いつでも保存されたデータを削除できます
  • 各サブスクの詳細画面の「解約・削除」で個別に削除できます
  • サブスク PRO の「バックアップ・引き継ぎ」機能で、お客様自身が JSON 形式でデータを書き出し・取り込みできます。書き出されたファイルはお客様が選んだ保存先に置かれ、開発者は関与しません
  • 開発者はお客様のデータにアクセスする手段を持っていません

5. 広告・トラッキングについて

本アプリは、広告を一切表示しません。広告識別子（IDFA）の利用や、他社アプリ・サイトをまたいだ行動追跡（トラッキング）は一切行いません。

アプリ改善のため、プロダクト分析サービス PostHog を用いて匿名の利用イベントを収集します（詳細は「1-3」参照）。これは個人を特定しない統計的な分析のみを目的とし、他社サービスをまたいだ広告トラッキングには使用しません。収集される ID はアプリが割り当てた匿名 ID であり、広告識別子（IDFA）ではありません。

6. OS 権限について

本アプリは、機能提供のため以下の OS 権限を利用します。許可は任意であり、許可しなくても基本機能は利用できます。

  • 通知（Notifications）：請求日のリマインド通知
  • 写真（Photo Library）：カスタムロゴ画像の選択（サブスク PRO 機能）

写真アクセスは、お客様が画像を選択した時のみ発生します。選択された画像は端末内のアプリデータに保存され、外部に送信されることはありません。

7. セキュリティ

  • 入力データはすべてデバイス上にのみ保存されます
  • 外部通信は本ポリシー「1-2」に記載のもののみで、いずれも HTTPS 通信を使用します
  • お客様を個人として識別できる情報を外部に送信することはありません

8. 本ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。重要な変更が生じた場合は、アプリのアップデートを通じてお知らせします。最新のポリシーは本ページ（サポートサイト）でご確認いただけます。

9. お問い合わせ

本プライバシーポリシーに関するご質問・ご意見は、開発者ホームページのお問い合わせ先からご連絡ください。
